Notify

Module: notify

Require: jquery, bootstrap

Params:

Attribútum Típus Érték Leírás
data-icon string fa-info-circle Bármelyik ikon a FontAwesome ikonkészletből. Lásd: FontAwesome
data-type string info Az alap Bootstrap típusok. Tehát: info, success, warning, danger.
data-url string false A felbukkanó notificationre kattintva lehet elérni a linket, amit itt megadsz.
data-blank boolean 0 A megadott link új ablakban nyíljon-e meg.
data-title string Figyelem! A felbukkanó értesítés címe.
data-message string null A felbukkanó értesítés szövege.
data-progress boolean false Legyen-e progress bar a felbukkanó értesítés alatt?
data-delay integer 5000 Az értesítés eltűnésének ideje.
data-timer integer 500 A progress bar frissítésének gyakorisága.
